# Emmanuel Kipyegon Portfolio

A modern, responsive personal portfolio website built with React and Vite to showcase my background, skills, experience, and featured projects.

## Overview

This portfolio highlights:

- My professional background as a software developer and ICT professional
- Key technical skills in software development, databases, and engineering tools
- Featured projects including a smart trolley system, a student career database, and this portfolio website itself
- A clean, modern UI designed for desktop and mobile devices

## Tech Stack

- React
- Vite
- CSS
- React Icons
- ESLint

## Project Structure

- src/components — page sections such as Hero, About, Skills, Projects, Experience, and Contact
- src/styles — modular stylesheet files for layout and responsive design
- src/assets — images and visual assets used across the site

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/manu601/emmanuel-portfolio.git
   ```

2. Navigate into the project directory
   ```bash
   cd emmanuel-portfolio
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

5. Open your browser and visit the local URL shown by Vite

## Available Scripts

- npm run dev — start the development server
- npm run build — create a production build
- npm run preview — preview the production build locally
- npm run lint — run ESLint checks
